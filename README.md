# Gemini AI Agent

A terminal-based AI Agent built using Google's Gemini 2.5 Flash model and Node.js.

## Features

* Interactive chatbot in terminal
* Gemini Function Calling
* Real-time weather information
* Mathematical calculations
* File reading capability
* Session memory
* Multi-tool reasoning (ReAct-style workflow)
* Extensible tool architecture

## Tech Stack

* Node.js
* Gemini 2.5 Flash
* Google GenAI SDK
* Axios
* Dotenv

## Installation

```bash
npm install
```

Create a `.env` file:

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Run the agent:

```bash
node index.js
```

## Example Queries

* What is the weather in Delhi?
* Calculate 25 * 30 + 100
* What time is it?
* Read file C:\Users\user\notes.txt
* What is the weather in Mumbai and what time is it?
